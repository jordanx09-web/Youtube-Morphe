## üì• Installation Guide

1.  **Download:** Get the latest `.zip` file from the [Releases Page](https://github.com/jordanx09-web/Youtube-Morphe/releases).
2.  **Flash:** Open **Magisk** or **KernelSU**, go to the *Modules* tab, and tap *Install from Storage*.
3.  **Select:** Choose the `YouTube.Morphe-v...zip` file.
4.  **Reboot:** Restart your device to apply the changes.

> **üí° Pro Tip:** If the app asks you to choose between "Stable" or "Experimental" on first launch, select **"Keep use experimental"**.
